# Enterprise Retail Analytics Platform

## Project Overview

This project simulates a real-world enterprise Business Intelligence solution using SQL Server, Power BI, and the Microsoft BI stack.

The objective is to design and build an end-to-end analytics platform, including:

- Business Requirements
- SQL Server Data Warehouse
- ETL Process
- Star Schema
- Power BI Data Model
- DAX Measures
- Dashboards
- Row-Level Security
- Power Automate Integration
- Documentation
- Testing

---

## Technology Stack

- SQL Server Developer Edition
- SQL Server Management Studio (SSMS)
- Power BI Desktop
- Power Query
- DAX
- Git
- GitHub
- Visual Studio Code
